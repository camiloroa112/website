Proyectos

Los siguientes son algunos de los trabajos que he realizado:

Efficient Frontier

Proyecto desarrollado en Python que muestra el grafico de frontera eficiente, que representa la maximización de utilidades sujeto a su riesgo, permitiendo identificar el punto optimo en el que con una combinación de activos se obtiene una mayor rentabilidad en base al riesgo asumido en la operación.

Código fuente:

Python
Pandas
Matplotlib
Jupyter

Mi Página Web

Presentación del código fuente y el repositorio público en GitHub donde se encuentran los archivos utilizados en el desarrollo de la presente página web. La página dispone de diseño responsivo con Bootstrap, dinamismo en la aparición de las etiquetas al cargar la página utilizando Ajax y estilos con CSS3.

Código fuente:

HTML
Bootstrap
CSS3
PHP

AMP Exporter

Proyecto que automatiza la creación de tablas y la ejecución de consultas en Snowflake a partir de instrucciones ingresadas dentro de un diccionario en Jupyter Notebook. El programa escribe las consultas, genera reportes en Excel, dispone de funcionalidades adicionales para ejecutar tareas especificas.

Código fuente:

Python
Snowflake
Jupyter

Portfolio

Ejercicio propuesto por la academia DePC Suite, poniendo en práctica los conocimientos adquiridos en herramientas como: HTML5, CSS3 y Bootstrap replicando una página web teniendo en cuenta sus estilos, diseño responsivo, separación de los elementos por columnas y la semantica de las etiquetas.

Código fuente:

HTML
Bootstrap
CSS3

HVA Dashboard

Proyecto que automatiza la generación de reportes en Excel a partir de la ejecución de consultas en Snowflake desde Jupyter Notebook, siendo importados a Tableau. El Dashboard presenta información de alto valor sobre la demografía, intereses, ubicación, y otros perfiladores de una audiencia.

Código fuente:

Jupyter
Snowflake
Tableau

Upload Snowflake

Script desarrollado en Python, cuya finalidad consiste en subir diferentes archivos tipo Excel como tablas en Snowflake. El programa elimina caractéres especiales en los encabezados, ajusta los tipos de datos en cada columna a tipo SQL, garantizando su carga exitosa en una base de datos especifica.

Código fuente:

Python
Snowflake
Jupyter

Email Reply - Send

Script desarrollado en Python, que dentro de sus funciones esta el envio correos electrónicos de forma masiva con un texto configurado predefinidamente en HTML a través de Outlook. Los destinatarios son indicados en un archivo en Excel y posee también la opción de respuesta de un correo preservando su formato.

Código fuente:

Python
Outlook
Jupyter

ARL Fill Form

Script desarrollado en Python junto con la libreria Selenium, su función principal radica en automatizar el completado un formulario de seguridad laboral solicitado por el gobierno en una página web, posteriormente el programa ingresa y completa un formulario adicional en Sharepoint requerido por la empresa.

Código fuente:

Python
Selenium

Excel Cleanser

Script desarrollado en Python junto con la libreria Pandas, elaborado con el objetivo de realizar procesos de limpieza, orden y consolidación de diferentes reportes en Excel descargados de SAP. Dispone un serie de inputs que brindan guía al usuario sobre la instrucción deseada (consolidación o transformación) y del estado del proceso.

Código fuente:

Python
Pandas
